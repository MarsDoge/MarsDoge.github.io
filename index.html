如何创建github上面的个人免费博客

首先声明一下，本文是在5分钟 搭建免费个人博客的基础上对原文的补充，大多使用的还是原文的材料，只是个人觉得对于小白来说还是有一些不够完善的地方（亲身经历，哈哈），因此我在这里引用一下原文，如果涉及到侵权请联系我，我将及时删除。

三步完成免费个人博客搭建， 这是一篇小白也能看懂的文章，本文主要针对OS X ，Windows 除了软件安装方式和命令有些区别（装了git bash也一样），其他基本一样。

OK 开始正文

1.1注册

首先你需要注册一个Github账号，已有的可以直接进行1.2，注意username，这会影响到你的域名，你的域名将会是 username.github.io ，所以认真的取个名字吧。
http://upload-images.jianshu.io/upload_images/136678-a5b815963631f1bf.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240
注册过程可能需要验证你的邮箱，其他就不在赘述。

1.2 创建仓库

然后需要创建一个仓库(repository) 来存储我们的网站，点击首页任意位置出现的 New repository按钮创建仓库, Respository name 中的username.github.io 的username 一定与前面的Owner 一致，记住你的username下面会用到。

第一步就已经完成了，下面是安装。

Hexo 可以说是目前最流行的博客框架了，基于Nodejs，更多信息可以google，下面需要安装的工具包括 Git，Nodejs，Hexo。（Windows 用户自行搜索这些工具，直接安装即可，试过基本没啥问题）

2.1 安装Git

// 如果已安装HomeBrew 无需执行此行
$ /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

$ brew install git // 安装Git
